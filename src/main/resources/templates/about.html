About Car Health Check

Car Health Check is an intelligent web application developed as part of my engineering thesis. The main goal is to assist users in evaluating the technical condition of used vehicles using AI-driven diagnostics.

Project Goals

  • AI-powered vehicle diagnostics
  • Vehicle condition assessment based on user input
  • Car inspection history tracking
  • Simplified car-buying decisions

Technologies Used

  • Spring Boot - Backend API & logic
  • Thymeleaf - Frontend templating
  • React Native - Mobile app (future)
  • Machine Learning - AI-based analysis
  • MariaDB - Data storage

Future Enhancements

  • VIN-based vehicle history lookup
  • Integration with external automotive APIs
  • Mobile-friendly real-time diagnostics
